Weather Diary

**Автор:** [Глеб Рыбкин]

Описание
Приложение для ведения личного дневника погоды. Позволяет фиксировать дату, температуру, описание условий и наличие осадков. Данные сохраняются локально в формате JSON.

Функции
- Добавление записей с валидацией данных.
- Фильтрация по минимальной температуре.
- Автоматическое сохранение и загрузка из JSON.

Как запустить
1. Убедитесь, что у вас установлен Python 3.x.
2. Скачайте файл `weather_diary.py`.
3. Запустите его командой:
   ```bash
   python weather_diary.py
   ```

Примеры тестов
1. **Валидация:** Попробуйте ввести буквы в поле "Температура" — программа выдаст предупреждение.
2. **Фильтр:** Введите несколько записей (например, +5, +15, +20). В поле фильтра введите "10" и нажмите кнопку — останутся только записи выше +10.
